Features

🔍

Multi-authority discovery

Fan-out search across MangaUpdates (manga), AniList (manhwa), MangaDex (manhua), NovelUpdates + WuxiaWorld (novels), nhentai (hentai). Results deduplicated by authority precedence.

⬇️

Multi-source download

Chapters aggregated across sources — automatic fallback if one goes down. Explicit sources auto-routed for hentai titles.

🧩

Plugin system

Add new download sources without recompiling. Covers manga, manhwa, manhua, novels, and hentai doujinshi.

📖

Web reader

Paged or scroll mode for comics, prose mode for novels. Chapter navigation and reading progress.

👥

Multi-user

Per-user libraries with shared file storage and individual reading progress.

⚙️

Self-hosted

Single Docker Compose file. Runs on a NAS, Raspberry Pi, or any always-on box.
