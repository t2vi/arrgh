*ARRgh

Self-hosted manga manager, downloader, and reader. Runs on your NAS, Pi, or any home server.

Get started → Deploy with Docker
terminal 
# Pull and start
curl -O https://raw.githubusercontent.com/t2vi/arrgh/main/docker-compose.yml
docker compose up -d

# Open in browser
open http://localhost:8080

Features

🔍

MangaUpdates metadata

Consistent titles, covers, descriptions and tags from a single authoritative source.

⬇️

Multi-source download

Chapters aggregated across sources — automatic fallback if one goes down.

🧩

Plugin system

Add new download sources without recompiling. Browse and install community plugins from Settings.

📖

Web reader

Paged or scroll mode, chapter navigation, reading progress, novel font size control.

👥

Multi-user

Per-user libraries with shared file storage and individual reading progress.

⚙️

Self-hosted

Single Docker Compose file. Runs on a NAS, Raspberry Pi, or any always-on box.
